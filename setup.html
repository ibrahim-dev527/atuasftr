Site Setup

Connect your Supabase database, configure admin credentials, and get the site running in minutes. Follow the steps below carefully.

STEP 1 — Supabase Project

Create a Free Supabase Project

Supabase is the free database that stores your supporter registrations. You only need to do this once.

  1. Go to supabase.com and create a free account.
  2. Click New Project → Give it any name (e.g. atuas-quiz).
  3. Set a strong database password and choose the Singapore or any nearby region.
  4. Wait ~2 minutes for the project to spin up.
  5. Go to Project Settings → API to get your URL and anon key.
STEP 2 — Create Database Table

Run This SQL in Supabase

In your Supabase dashboard, go to SQL Editor → New Query and paste this exactly:

-- Create supporters table CREATE TABLE IF NOT EXISTS supporters ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, first_name TEXT NOT NULL, last_name TEXT NOT NULL, student_id TEXT NOT NULL, phone TEXT NOT NULL, year_level TEXT, programme TEXT, registered_at TIMESTAMPTZ DEFAULT NOW() ); -- Allow anyone to insert (for registration form) CREATE POLICY "Anyone can insert" ON supporters FOR INSERT WITH CHECK (true); -- Allow anyone to read (for admin page) CREATE POLICY "Anyone can read" ON supporters FOR SELECT USING (true); -- Allow anyone to delete (admin use) CREATE POLICY "Anyone can delete" ON supporters FOR DELETE USING (true); -- Enable Row Level Security ALTER TABLE supporters ENABLE ROW LEVEL SECURITY;

Click Run after pasting. You should see "Success. No rows returned."

STEP 5 — Add Your Images

Upload These Files to Your Site Folder

Place these image files in the same folder as your HTML files:

  1. flyer.jpg — Campaign/event flyer (shows as popup after 10s)
  2. ATU.jpg — Accra Technical University logo
  3. uew.jpg — University of Education, Winneba logo
  4. developer.jpg — Your photo (for the developer section)

All images can be .jpg, .png, or .webp. Recommended size: logos 200×200px, flyer 800×1000px.

STEP 6 — Deploy & Go Live

Deploy to Netlify (Free Hosting)

  1. Create a free account at netlify.com
  2. Drag and drop your entire site folder onto the Netlify dashboard.
  3. Your site goes live instantly with a free URL!
  4. Optionally, connect your own domain name.

Alternatively, upload to GitHub and connect to Netlify for automatic deploys.
